# Results

This folder contains all evaluation and benchmark outputs generated by running the full pipeline.

## How to Reproduce

```bash
# From the project root, with the backend venv active:
cd backend && source venv/bin/activate && cd ..
./scripts/run_complete_pipeline.sh
```

Or step by step:

```bash
# 1. Download 200-image COCO val2017 subset
python scripts/prepare_coco_subset.py --num-images 200

# 2. Export weights (run from backend/)
cd backend
python ../scripts/export_torchscript.py --model yolov8 --output weights/yolov8n.torchscript
python ../scripts/export_torchscript.py --model yolov5 --output weights/yolov5s.torchscript
python ../scripts/export_onnx.py        --model yolov8 --output weights/yolov8n.onnx
python ../scripts/export_onnx.py        --model yolov5 --output weights/yolov5s.onnx
cd ..

# 3. Run mAP evaluation
cd backend
python ../scripts/evaluate_dataset.py \
    --model yolov8 yolov5 --compare \
    --annotations ../data/annotations/instances_val200.json \
    --images-dir   ../data/images/val \
    --output       ../results/eval_report.csv
cd ..

# 4. Run latency benchmark
cd backend
python ../scripts/benchmark_models.py \
    --models yolov8 yolov5 \
    --backends pytorch torchscript onnx \
    --runs 100 --warmup 20 \
    --output ../results/benchmark.csv
cd ..

# 5. Run video inference
cd backend
python ../scripts/run_video_inference.py \
    --generate-test-video \
    --compare --max-frames 150 \
    --results-dir ../results \
    --output-dir  ../outputs
cd ..
```

## Files in This Directory

| File | Contents |
|------|----------|
| `eval_report.csv` | mAP@0.5, mAP@0.5:0.95, FPS per model/backend |
| `benchmark.csv` | Latency avg/min/max/std + FPS per model/backend |
| `benchmark.json` | Same as CSV but machine-readable |
| `video_benchmark.csv` | Video inference FPS + latency summary |
| `video_benchmark.json` | Detailed per-frame data |
| `predictions_last.json` | Raw COCO-format predictions from last eval run |
